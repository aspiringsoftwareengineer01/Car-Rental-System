# Antigravity Car Rental System 🚗💨

A premium, state-of-the-art Car Rental Web Application designed with a sleek, modern UI, glassmorphism aesthetics, dynamic micro-animations, and a highly responsive user experience.

## 🛠️ Technology Stack

- React JS
- Tailwind CSS
- Supabase
- Vercel

## Features

- User Authentication
- Car Listings
- Car Booking
- Admin Dashboard
- Responsive Design
- Premium UI/UX

## Project Goal

To create a modern online platform where customers can browse, book, and manage car rentals easily.
